----------------------------------------------------Python Setup----------------------------------------------------------------------------------------------------------------------------

1 Download latest version of python from official website of the python

2 Set below system variables
a. C:\Users\neham\AppData\Local\Programs\Python\Python312
b. C:\Users\neham\AppData\Local\Programs\Python\Python312\Scripts

-------------------------------------------------selenium download--------------------------------------------------------------------------------------------------------------------------
1. Open the command prompt and execute below command: pip install selenium

2. For verification of the selenium installation execute: pip show selenium
